Quelques projets

Voici des projets que j'ai réalisé dernièrement. N'hésitez pas à y jeter un oeil !

Jeu Fruit Ninja Slicer

Python & Pygame

Création d'un jeu en python avec pygame. Les fonctionnalités demandées étaient de jouer, quitter, sélectionner la langue et la difficulté, ajouter une musique de fond et pouvoir la mettre en pause. Le jeu lance des fruits de façon aléatoire avec une lettre, si une lettre est présente plusieurs fois, il y a un combo. Des galçons, gélent le temps pendant 5s, les bombes font perdre. A chaque fruit non coupé on perd une vie.

Jeu du pendu

Python & Pygame & json

Création d'un jeu du pendu en python avec pygame. Les fonctionnalités demandées étaient de jouer, quitter, sélectionner la difficulté. Le jeu prend des mots de façon aléatoire dans un fichier json. Les lettres entrées qui se trouvent dans le mot remplace _ par la lettre, sinon s'affiche comme lettre déjà entrée et ne peut plus être entrée. On gagne si le mot est trouvé avant que le pendu soit complet, sinon on perd. On peut ajouter de nouveaux mots.

Jeu du morpion

Python

Création d'un jeu du morpion. Les fonctionnalités demandées étaient de choisir le nom des joueurs et les symboles en les affectant à chaque joueur. Les cases sont vides au début, on choisi une case, puis au tour de l'adversire tout en vérifiant que la case est libre, sinon le joueur doit choisir une nouvelle case. Si la grille est remplis sans gagnant c'est une égalité, sinon, si 3 cases sont remplies en ligne, en colonne ou en diagonale par un même symbole, le joueur dont le symbole est attribué gagne.

Horloge avec alarme

Python

Développement d'une horloge avec comme fonctionnalité, possibilité de mettre une alarme. Si l'alarme est réglée, elle renvoie un message lorsque l'heure est égale à celle de l'alarme. Possibilité de régler l'heure, si non, l'heure démarre à 00:00:00, sinon on règle l'heure. Possibilité d'arrêter l'heure, puis de la relancer. Choisir entre le format 24h ou 12h avec AM/PM. Gestion des erreurs.

Calculatrice

Python

Développement d'une calculatrice avec comme fonctionnalité, afficher le menu. Commencer un nouveau calcul et renvoyer le résultat. Continuer le calcul à partir du résultat précédent. Stocker les opérations dans un fichier json. Afficher l'historique, le supprimer. Sortir du programme. Gestion des erreurs.
